bool Emojidex::Service::Settings::closed_net
static

Special flag to specify that endpoints are on a closed network, not the official service.

As using closed_net functionality generally violates the emojidex Open License for many use cases this feature isn't actually implemented. In the future if will be implemented if there is enough demand from closed-source and commercial license holders.
