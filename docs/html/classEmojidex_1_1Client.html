Detailed Description

Core client class (includes all components in a central state-machine client)

In general you'll want to run everything through a client as long as you are implementing something with a single user. Multiple client instances could produce unpredicable behaviour and should be avoided. By loggin in with the User instance of this client class user details will automatically be set on appropriate queries.
